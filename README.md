# splatoon-bot-2

本项目使用 Deno 运行时, 重写了原 Node.js 项目 [splatoon2-qqbot](https://github.com/spacemeowx2/splatoon2-qqbot)

重写主要目标是能够支持 Splatoon3 的信息查询.

## 依赖

[go-cqhttp](https://github.com/Mrs4s/go-cqhttp).
